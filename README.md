# 3Skill Internship Projects

This repository serves as a portfolio of technical projects and implementations developed during my internship at **3Skill**. The projects focus on building responsive web applications, implementing front-end logic, and applying software engineering principles to create scalable solutions.

## 🚀 Projects Overview

### 1. BingeBox
A modern media discovery platform designed with a focus on user experience and clean aesthetics.
* **Core Features:** Dynamic content display and responsive layout.
* **Design Philosophy:** Minimalist, high-quality interface with a focus on dark-mode compatibility.
* **Tech Stack:** HTML5, CSS3.

### 2. BingeBox_js
The interactive engine behind the BingeBox application, handling the logical layer of the project.
* **Core Features:** Dynamic DOM manipulation and event-driven interactivity.
* **Functionality:** Handles media filtering and interactive elements using vanilla JavaScript.
* **Tech Stack:** JavaScript (ES6+).

## 🛠️ Skills & Technologies
* **Languages:** HTML5, CSS3, JavaScript (ES6+)
* **Development Principles:** Responsive Web Design, Component-based Logic, SDLC
* **Tools:** Git, GitHub, VS Code
